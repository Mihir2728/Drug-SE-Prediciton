Drug–Gene Network Reveals Biological Determinants of ADR Predictability
This repository contains the code, data pipeline, and analysis for an MSc thesis investigating adverse drug reaction (ADR) prediction using a drug similarity network built from experimentally validated drug–gene interactions. A harmonic function propagation model predicts ADR risk across a broad panel of toxicity phenotypes, and KEGG pathway / Gene Ontology enrichment analysis is used to explain why prediction performance varies so much across different ADRs — linking computational predictability back to the underlying biological network architecture.
Key components:

Harmonic label propagation model for ADR prediction over drug–gene interaction graphs
Five-fold cross-validation and per-ADR performance ranking
KEGG and GO enrichment analysis comparing highly predictable vs. poorly predictable ADRs
Supporting figures and tables for the accompanying manuscript
